# Equilibrium

**Only you can bring balance to the Network!**

![Equilibrium screenshot](/screenshot.png)

## instructions

**Goal** - Ensure all nodes are non-negative.

**Controls** -

1. Click or tap on a node to send one point of value to each neighbor.

2. Right-click or long-press on a node with a single neighbor to send all of its value to that neighbor.
